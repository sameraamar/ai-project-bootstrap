# Project Starter Instructions

This repository is not an application or runnable project.

Its purpose is to hold and version the main instruction document used for starting new software projects with an AI-first, documentation-first workflow.

The primary file in this repository is:

- `project-starter-instructions.md`

That file defines the guidance for bootstrapping a new project, including:

- the optional information to collect at the start of a project
- the initial documentation structure to generate
- the rules for keeping documentation as the source of truth
- the expected contents of the starter documentation files

Use this repository as a stable place to maintain and update those instructions over time.
